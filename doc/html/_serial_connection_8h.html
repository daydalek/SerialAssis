Detailed Description

This class is responsible for the serial connection.

Note
This class is a wrapper of QSerialPort,including sending data, sending file, receiving data, also there's a Errorcode class named SerialConnectionState ,which can be passed to MainWindow to do some specific operations,such as pop up a window to warn users that serialport is not opened.
