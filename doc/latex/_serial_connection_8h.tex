\doxysubsection{Detailed Description}
This class is responsible for the serial connection. 

\begin{DoxyNote}{Note}
This class is a wrapper of QSerial\+Port,including sending data, sending file, receiving data, also there\textquotesingle{}s a Errorcode class named Serial\+Connection\+State ,which can be passed to \doxylink{class_main_window}{Main\+Window} to do some specific operations,such as pop up a window to warn users that serialport is not opened. 
\end{DoxyNote}
